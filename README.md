# XYZ Bookstore

A modern, professional bookstore website featuring a beautiful UI, smooth animations, and comprehensive e-commerce functionality. The website provides an exceptional shopping experience for academic and professional books across multiple engineering disciplines.

## Features

### User Experience
- **Modern Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Polished transitions and hover effects throughout the site
- **Search Functionality** - Real-time search across books, authors, and categories
- **Category Filtering** - Easy filtering by department (CSE, ECE, EEE, Civil)

### Shopping Features
- **Shopping Cart** - Full-featured cart with quantity management and total calculation
- **Wishlist** - Save favorite books for later purchase
- **Book Details** - Quick view modal with comprehensive book information
- **Ratings & Reviews** - Display book ratings and review counts

### Design Elements
- **Hero Section** - Eye-catching landing page with statistics and call-to-action buttons
- **Category Cards** - Interactive department cards with icons and descriptions
- **Featured Books Grid** - Dynamic book display with hover effects
- **Features Section** - Highlight key benefits (fast delivery, secure payment, easy returns, 24/7 support)
- **About Section** - Company information and statistics
- **Footer** - Comprehensive footer with quick links, customer service, and newsletter signup

### Technical Features
- **Modal System** - Professional modals for cart, wishlist, authentication, and book details
- **Local Storage** - Persistent cart and wishlist data
- **Smooth Scrolling** - Seamless navigation between sections
- **Scroll-to-Top Button** - Convenient navigation for long pages
- **Mobile Menu** - Responsive hamburger menu for mobile devices

## Technology Stack

- **HTML5** - Modern semantic markup
- **CSS3** - Custom styling with CSS variables, flexbox, and grid
- **JavaScript (ES6+)** - Vanilla JavaScript with modern features
- **Vite** - Fast build tool and development server
- **Font Awesome** - Icon library for UI elements

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```
